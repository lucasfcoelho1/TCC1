\chapter{Proposta}  

Com base no conteúdo apresentado no \chaptername 2, o presente trabalho propõe verificar a eficácia da ferramenta x-PATeSco, com o intuito de analisar se os \textit{scripts} de testes gerados conseguem se adaptar a mudanças entre as versões de uma mesma \textit{app}.
